# NNI_Sample
UE5のNeuralNetworkInferenceプラグインを使用した手書き数字認識と物体検出のサンプルプロジェクトです。

### 構成

使用エンジンバージョン UE5.0.3

##### 手書き数字認識

- /Game/MNIST_SampleMap

##### 物体検出

- /Game/YOLOv7_SampleMap
